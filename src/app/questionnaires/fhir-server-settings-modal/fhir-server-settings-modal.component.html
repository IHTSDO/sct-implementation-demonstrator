FHIR Server Connection

Manager mode allows you to store questionnaires in a FHIR Server. Enter the FHIR server URL and your User Tag — this value will be used to identify your questionnaires.

warning

Using a public FHIR server does not guarantee persistence. Always save your questionnaires locally as a backup.
